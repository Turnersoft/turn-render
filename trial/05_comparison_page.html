Groups
Definition

A group is a set G equipped with a single binary operation * that satisfies four axioms.

Required Axioms

1. Closure

For all a, b ∈ G, we have a * b ∈ G

2. Associativity

For all a, b, c ∈ G, we have (a * b) * c = a * (b * c)

3. Identity Element

There exists e ∈ G such that e * a = a * e = a for all a ∈ G

4. Inverse Elements

For each a ∈ G, there exists a⁻¹ ∈ G such that a * a⁻¹ = a⁻¹ * a = e

Properties

Single Operation

Groups have exactly one binary operation

Commutativity

Not required - groups can be non-abelian

Unique Identity

The identity element is unique

Examples

Integers under Addition

(ℤ, +) with identity 0

Symmetric Group

S_n - permutations under composition

Matrix Group

GL(n, ℝ) - invertible matrices under multiplication

Key Insight: Groups focus on symmetry and transformations with a single, well-behaved operation.
Rings
Definition

A ring is a set R equipped with two binary operations + and ⋅ that satisfy multiple axioms.

Required Axioms

Additive Structure

Closure, Associativity, Identity (0), and Inverses for +

Plus: Addition is commutative

Multiplicative Structure

Closure and Associativity for ⋅

No multiplicative identity required (unless specified)

No multiplicative inverses required

Distributive Laws

a ⋅ (b + c) = a ⋅ b + a ⋅ c

(a + b) ⋅ c = a ⋅ c + b ⋅ c

Properties

Two Operations

Rings have exactly two binary operations that interact

Commutativity

Required for addition, optional for multiplication

Multiple Identities

Unique additive identity (0), possibly a multiplicative identity (1)

Examples

Integers

(ℤ, +, ⋅) with identities 0 and 1

Polynomial Ring

R[x] - polynomials over a ring R

Matrix Ring

M_n(R) - n×n matrices over a ring R

Key Insight: Rings model arithmetic-like structures where addition and multiplication interact via distributivity.
