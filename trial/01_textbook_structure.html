Chapter 1: Introduction to Groups
1.1 Basic Definitions

In this section, we introduce the fundamental concept of a group, which is one of the most important algebraic structures in mathematics.

Definition 1.1 (Group)

A group is a set G equipped with a binary operation * that satisfies the following axioms:

  1. Closure: For all a, b ∈ G, we have a * b ∈ G
  2. Associativity: For all a, b, c ∈ G, we have (a * b) * c = a * (b * c)
  3. Identity: There exists an element e ∈ G such that e * a = a * e = a for all a ∈ G
  4. Inverse: For every a ∈ G, there exists a⁻¹ ∈ G such that a * a⁻¹ = a⁻¹ * a = e
Theorem 1.2 (Uniqueness of Identity)

The identity element in a group is unique.

Proof:

Suppose e₁ and e₂ are both identity elements. Then:

e₁ = e₁ * e₂ = e₂

where the first equality uses the fact that e₂ is an identity, and the second equality uses the fact that e₁ is an identity. ∎

Example 1.3 (Integers under Addition)

The set of integers ℤ with the operation of addition forms a group:

  • Closure: The sum of two integers is an integer
  • Associativity: (a + b) + c = a + (b + c) for all integers
  • Identity: 0 is the additive identity
  • Inverse: For any integer a, its inverse is -a
1.2 Group Properties
1.2.1 Basic Properties

We now explore some fundamental properties that follow from the group axioms.

Theorem 1.4 (Cancellation Laws)

In any group G, the following cancellation laws hold:

  1. Left cancellation: If a * b = a * c, then b = c
  2. Right cancellation: If b * a = c * a, then b = c

Example 1.5 (Symmetric Group)

The symmetric group S_n consists of all permutations of n elements, with composition as the group operation.
