Theorem: Uniqueness of the Identity Element in a Group

Theorem 1.1
Let G be an arbitrary group. If e₁ is an identity element in G, and e₂ is an identity element in G, then e₁ = e₂.
Definition (Group G): A set with a binary operation *G satisfying group axioms.
  • Operation: * (denoted *G when specific to G)
  • An identity element e exists such that for all x in G, e * x = x * e = x. (Axiom: ID_Axiom)
  • For each element x in G, an inverse inv(x) exists.
  • Axiom Set: StandardGroupAxioms_v1.2
Proof.
H1 (Hypothesis 1): e₁ is an identity element in G.
This implies by ID_Axiom: ∀ x ∈ G, (e₁ * x = x) and (x * e₁ = x).
H2 (Hypothesis 2): e₂ is an identity element in G.
This implies by ID_Axiom: ∀ y ∈ G, (e₂ * y = y) and (y * e₂ = y).
Goal (Target): Show that e₁ = e₂.
1. Consider the element e₁. Since e₁ ∈ G and e₂ is an identity element in G (by H2), we can apply the right identity property of e₂ to e₁.
F1 (Fact 1): e₁ = e₁ * e₂.
Tactic: apply ID_Axiom (right identity part of H2) with y := e₁.
2. Next, consider the expression e₁ * e₂. Since e₂ ∈ G and e₁ is an identity element in G (by H1), we can apply the left identity property of e₁ to e₂.
F2 (Fact 2): e₁ * e₂ = e₂.
Tactic: apply ID_Axiom (left identity part of H1) with x := e₂.
3. Combining Fact 1 (e₁ = e₁ * e₂) and Fact 2 (e₁ * e₂ = e₂), by the transitivity of equality, we conclude:
Conclusion: e₁ = e₂.
Tactic: transitivity on F1, F2.

Final Proof State

Goal (Target): e₁ = e₂   (Proven)
Q.E.D.
